# 🛠️ TunaMentoora.ai Teknoloji Yığını (Tech Stack)

Projenin geliştirilme sürecinde modern, hızlı ve yapay zeka odaklı araçlar tercih edilmiştir:

- *Dil & İskelet:*
  - *HTML5 & CSS3:* Modern, duyarlı (responsive) ve kullanıcı dostu bir arayüz için.
  - *JavaScript (Vanilla):* Hızlı çalışma ve API entegrasyonu için saf JavaScript kullanılmıştır.
- *Yapay Zeka (AI):*
  - *Google Gemini 1.5 Flash API:* Uygulamanın zekasını oluşturan bu model, kullanıcı verilerini analiz ederek kişiselleştirilmiş kariyer rotaları üretmektedir.
  - *​Cursor (AI-Native Code Editor):* Geliştirme sürecinde yapay zeka yardımıyla kod yazımını hızlandırmak ve karmaşık yapıları düzenlemek için kullanılmıştır.
  - *​Google AI Studio:* Yapay zeka modelinin davranışlarını test etmek ve API anahtarını güvenli bir şekilde yönetmek amacıyla tercih edilmiştir.
  - *​GitHub Copilot:* Kod yazarken anlık öneriler almak ve teknik hataları en aza indirerek verimliliği artırmak için sürece dahil edilmiştir. 
- *Geliştirme Ortamı:*
  - *Cursor (Agent Mode):* Vibe Coding yaklaşımıyla, yapay zeka ile iş birliği içinde kod geliştirilmiştir.
- *Versiyon Kontrol:*
  - *GitHub:* Kodun saklanması ve Buildathon süreci yönetimi için.
- *Yayınlama (Deployment):*
  - *Netlify:* Uygulamanın dünyaya açılması için kullanılan bulut platformları.
